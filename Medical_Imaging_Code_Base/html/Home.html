Train

The training page allows you to try your hand at identifying cancerous tumors from chest x-rays. If you are a first time user, you will need to create an account before training to save your progress. First time users may also find the “Information” tab to be helpful in understanding how perceptual learning works.

Information

To learn about lung health and perceptual learning go to the information tab. MITA uses perceptual learning to train you on how to read x-ray images. MITA focuses specifically on chest x-rays, so you can learn about lung health on this page as well.
